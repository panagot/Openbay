# Feedback request: Virtual DeCharge World on peaq (grant application)

We are applying for a **peaq ecosystem grant** in the **$10,000–$20,000** range and would value your critical feedback and recommendations before finalizing our submission.

---

## What we’ve built

**Virtual DeCharge World** — *Turn any parking spot into a revenue-generating EV charging node.* We are building the **operating system for independent EV charger owners**, powered by [peaq](https://www.peaq.xyz/) (Machine Economy, EVM-compatible).

- **Open coordination layer** — Not just an app: any charger can register, expose availability, and be discovered by drivers or third-party apps. We will expose a **public API/SDK** for charger discovery, pricing, and session data. Charger Registry (on peaq) + Session Events (on-chain or verifiable) + Public API/SDK (planned).
- **Grid + Map** — Grid = configurable garage/lot layout (mint, deploy, set rates, per-second settlement + POINTS). Map = real-world view (Google Maps heatmap, markers, session cards). “Load demo for judges” spawns example stations (e.g. Bengaluru, Delhi, Mumbai).
- **peaq integration** — MetaMask, peaq agung testnet, **peaq ID** per charger. Roadmap: peaq verify (Trust Score), peaq pay (P2P), ERC-20 POINTS.
- **Entrepreneur angle** — Real yield from physical chargers; without peaq ID, chargers can’t be trusted, discovered, or monetized in a decentralized way. Illustrative: a garage with 2 chargers 6h/day could generate on the order of €100–300/month (location-dependent), 100% to owner via peaq pay.
- **Path to real infrastructure** — Phase 1: simulated + peaq ID (current). Phase 2: EV charger APIs. Phase 3: IoT (OCPP, ESP32/RPi). Phase 4: peaq pay + composability with other peaq DePINs.
- **Differentiation** — Centralized networks (ChargePoint, PlugShare) = directories, closed, high fees. We = permissionless onboarding, machine-native identity (peaq ID), P2P payments; we are an **ownership layer** that empowers hosts. Hardware-agnostic (retrofit existing chargers).
- **Grant ask** — Deliverables: ERC-20 POINTS, peaq verify, public charger registry, at least one real-world data source (API or IoT). Budget sketch: ~$7K dev, ~$5K marketing/pilots, ~$4K testing/audit, ~$4K misc.

**One-liner for reviewers:** *As EV adoption grows, millions of underutilized parking spaces will become micro-infrastructure nodes. Virtual DeCharge World positions peaq as the coordination layer for this emerging machine economy.*

**Live demo:** [https://powering-the-e-volution-on-peaq.vercel.app/](https://powering-the-e-volution-on-peaq.vercel.app/)

**Suggested reviewer flow:** Connect wallet (or “Local dev” wallet) → **“Load demo for judges”** in Charge Points panel → switch to **Map** tab. Use Grid to mint/deploy/register on peaq.
